## Quick Start

### Claude Code

```bash
# Register marketplace
/plugin marketplace add frxiaobei/frxiaobei-skills

# Install skill
/plugin install youyou@frxiaobei-skills
```

### OpenClaw

```bash
clawhub install youyou
```
