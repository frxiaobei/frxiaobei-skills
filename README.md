## Quick Start

### Claude Code

```bash
# Register marketplace
/plugin marketplace add frxiaobei/frxiaobei-skills

# Install skill
/plugin install youyou-health@frxiaobei-skills
```

### OpenClaw

```bash
# Install voice notes
clawhub install elyfinn-voice-notes

# Install health manager
clawhub install youyou-health

# Install image downloader
clawhub install article-image-downloader
```
